Portal da Zeladoria - Análise de Dados e CI

Período: Fevereiro 2025 - Junho 2025

Desenvolvimento do Portal da Zeladoria, solução web construída em equipe com foco em coleta, modelagem e visualização de dados para apoio à gestão pública. O projeto evoluiu a partir de versões anteriores e, nesta etapa, teve como objetivo central ampliar a capacidade de análise estratégica por meio de ferramentas de BI. O projeto utiliza Flask (Python) para o backend, seguindo o padrão RESTful e retornando dados em formato JSON, e frontend com HTML, CSS e JavaScript, com foco em acessibilidade – incluindo integração com a API VLibras. A visualização dos dados foi realizada com Power BI, permitindo a geração de dashboards dinâmicos para auxiliar decisões da gestão municipal.

Contribuições:

  • Elaboração do relatório técnico do projeto, com fundamentação metodológica baseada em Engenharia de Software e SCRUM.
  • Criação da base de dados fictícia utilizada no Power BI, alinhada aos dados manipulados pela API.
  • Implementação da integração contínua (CI) com GitHub Actions, desenvolvendo os arquivos .yml para validação automatizada de código HTML e JS.
  • Configuração e acompanhamento da pipeline de desenvolvimento, garantindo que os commits e pull requests fossem automaticamente verificados e organizados.
  • Apoio aos demais integrantes com suas tarefas técnicas e conceituais, garantindo coesão entre as etapas de frontend, backend e análise de dados.
